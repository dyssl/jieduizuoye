# 运行环境
操作系统：windows；python版本：3.7，需安装flask，Flask-Cors，itertools模块
# 编译和使用
1.后端源代码在AutoPlayForShisanshui文件夹中，可下载后在上述运行环境中运行

2.其余部分全部为前端文件，全部下载后用火狐或谷歌浏览器打开主页.html文件即可查看所有的UI实现

3.也可以在releases中下载defaul.exe文件，下载完成直接打开即可使用
